【安心してお使いいただくために】

🛡 このアプリには危険な処理やウイルスなどは一切含まれておらず、
Pythonソースコード（app.py）もすべて同梱されています。

⚙ 初回実行時には、必要なライブラリ（flask, PyPDF2, Pillow）を
自動的にインストールします（インターネット接続が必要です）。

🧩 セキュリティの都合でmacOSによりブロックされることがありますが、
.command ファイルは自己修復処理（自動解除）を行いますのでご安心ください。

📍もし開けない場合は：

① launch_flask.command を右クリック →「開く」  
② 開かない場合は以下のターミナルコマンドを1回だけ実行してください：

```bash
xattr -d com.apple.quarantine launch_flask.command
